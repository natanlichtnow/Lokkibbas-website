# Demo para GitHub Pages

Esta pasta `docs/` e uma versao estatica do sistema para demonstracao ao cliente.

## Funcionalidades da demo
- Area admin com login
- Consulta publica por serial
- Layout do laudo igual ao sistema principal
- Calculo de porosidade media
- Status de manutencao
- Galeria com popup (abrir, fechar por fundo, `Esc`, `x`)
- Download em PDF via impressao

## Publicar
1. Suba este projeto para o GitHub.
2. Abra `Settings > Pages`.
3. Em `Build and deployment`, selecione:
- Source: `Deploy from a branch`
- Branch: `main` (ou a branch que usar)
- Folder: `/docs`
4. Salve e aguarde o link de publicacao.

## Importante
- Esta versao usa `localStorage` no navegador (sem backend).
- Os dados da demo ficam no navegador de quem estiver usando a pagina.
